# =========================
# Installation Instructions
# =========================

# 1. Create virtual environment:
#  
#   python -m venv venv
#
# 2. Activate virtual environment:
#    Windows: venv\Scripts\activate
#    Unix/macOS: source venv/bin/activate
#
# 3. Install dependencies:
#    pip install -r requirements.txt
#
# 4. Run application:
#    streamlit run app.py

# =========================
# Troubleshooting
# =========================

# If you encounter issues:
#
# 1. Update pip:
#    pip install --upgrade pip
#
# 2. Install wheel:
#    pip install wheel
#
# 3. Try installing problem packages individually:
#    pip install numpy
#    pip install pandas
#    (etc.)
#
# 4. For Apple Silicon (M1/M2) Macs:
#    Use conda instead of pip for better compatibility:
#    conda install numpy pandas scipy statsmodels plotly
#    pip install streamlit
#
# 5. Clear pip cache if needed:
#    pip cache purge

# =========================
# Version Lock File
